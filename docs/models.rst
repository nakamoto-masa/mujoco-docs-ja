MuJoCo Menagerie
----------------

物理シミュレーターは、シミュレートするモデルの品質に依存します。MuJoCoのような多くのモデリングオプションを持つ強力なシミュレーターでは、期待通りに動作しない「悪い」モデルを簡単に作成してしまう可能性があります。 **MuJoCo Menagerie** の目標は、すぐにうまく動作する、適切に設計されたモデルの厳選されたコレクションをコミュニティに提供することです。

詳細については、 `Menagerie repository <https://github.com/google-deepmind/mujoco_menagerie>`__ を参照してください。
